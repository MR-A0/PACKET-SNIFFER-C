## INSTALLATION [root@A:-] :

```
apt-get update -y
```
```
apt-get upgrade -y
```
```
sudo apt-get install libpcap-dev 
```
```
git clone --depth 1 https://github.com/MR-A0/PACKET-SNIFFER-C.git
```


## EXECUTION [root@A:-] :

```
cd PACKET-SNIFFER-C
```
```
chmod 777 PACKET_SNIFFER
```
```
sudo ./PACKET_SNIFFER eth0
```
